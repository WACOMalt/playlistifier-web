What does this tool do?
This Tool uses the Spotify Web Data API to look up your playlists and create a list of the tracks. It then does its best to search for those songs on YouTube. The tool then provides a list of individual YouTube URLs which can be used by the user. The tool also makes it easy to convert a Youtube Playlist into an easy list of individual video URLs. It does this using the official APIs for each service. In practice, this tool enables nothing a user of each service couldn't already do manually, it just makes it much easier!
Using our built in converter, users can take their downloaded video file and convert and tag the file to a high quality MP3 using the data from spotify.
How do I use the "Download to Folder" feature?
The "Download to Folder" button allows you to save files directly to a folder on your computer without creating a ZIP file. This feature is only available in supported browsers:

How to use:
1. Download your tracks first using the "Download All Tracks" button

4. The app will save all your music files directly to the chosen folder
5. Files will be automatically numbered and named with artist and track information
Why is "Download to Folder" disabled in my browser?
The "Download to Folder" feature requires the File System Access API, which is not supported in all browsers:
Firefox & Safari: These browsers don't support the File System Access API for security reasons. Use the "Download All Tracks" button instead, which will create a ZIP file you can extract.
Alternative for unsupported browsers:
1. Use "Download All Tracks" to get a ZIP file
2. Extract the ZIP file to your desired location
3. The extracted files will be organized exactly the same way

How do I get my music onto an MP3 player or phone?
Method 1 - Direct Folder (Chrome/Edge/Opera):
1. Connect your MP3 player or phone to your computer
2. Use "Download to Folder" and select your device's music folder
3. Files will be saved directly to your device
4. Safely disconnect your device
Method 2 - ZIP Download (All browsers):
1. Use "Download All Tracks" to get a ZIP file
2. Extract the ZIP file on your computer
3. Copy the extracted files to your MP3 player or phone
4. The files will be properly named and organized
